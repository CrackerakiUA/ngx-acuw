Installation

1. Install three.js via npm:

npm install three

2. Install ACUW Library

npm install ngx-acuw

Particle-As-Images API

Properties

Name Default value Description
@Input() imageUrls: string null URL to the image
@Input() backgroundColor: string #222222 The background-color CSS property sets the background color of an element.
@Input() imageWidth: string 100% Width of the generated image in "px" or "%"
@Input() imageHeight: string 100% Height of the generated image in "px" or "%"
@Input() transitionDuration: number 1000 Duration of the transition effenct in ms
@Input() horizontalAlignment: string center Possible values: "start", "center", "end"
@Input() verticalAlignment: string center Possible values: "top", "center", "bottom"
@Input() animationEnabled: boolean true The animation can be stopped by setting the property to "false", e.g. if the element is not in viewport anymore.
